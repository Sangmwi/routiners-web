# 🪖 루티너스 회원가입 & 내 프로필 UX 플로우 (MVP)

> **루티너스는 현역 군인을 위한 서비스입니다.  
> 몇 가지만 확인하면 바로 사용할 수 있어요.**

---

## 1. 회원가입 전체 흐름 (요약)

```
Google 로그인
→ PASS 실명 인증 (임시 인증 상태)
→ 군인 정보 입력 (단계형)
→ 가입 완료 = User 생성
```

- ❌ 가입 후 인증
- ❌ 미완 계정 상태
- ⭕ 모든 단계 완료 시에만 User 생성

---

## 2. Step 1. Google 로그인 (진입 게이트)

### 목적
- 계정 식별
- OAuth 기반 기본 신뢰 확보

### 수집 정보
```ts
providerId
email
name? // optional
```

> ⚠️ 이 단계에서는 **User 생성하지 않음**  
> → 임시 인증 상태(Session only)

---

## 3. Step 2. PASS 실명 인증 (필수)

### UX
- 버튼: **[ PASS로 본인 확인 ]**
- 성공 시 → **다음 단계로 자동 이동**

### 수집 정보 (UI 비노출)
```ts
realName
phoneNumber
birthDate
gender
```

---

## 4. Step 3. 군인 정보 입력 (단계형 · 필수)

### 공통 안내 멘트
> **루티너스는 현역 군인을 위한 서비스예요.  
> 몇 가지만 확인하면 바로 사용할 수 있어요.**

### 4-1. 입대 시기
- Select (YYYY / MM)
- 논리 범위 검증

### 4-2. 현재 계급
- 이병 · 일병 · 상병 · 병장 (+ 호봉)
- 입대 시기와 논리 검증

### 4-3. 소속 부대
- 검색 기반 선택 (JSON 상수)
- 자유 입력 불가

### 4-4. 병과
- Select 방식

### 4-5. 닉네임
- Input + 중복 체크
- 실명은 공개되지 않음

---

## 5. Step 4. 최종 확인

```
- 닉네임: 군돌이123
- 입대: 2025년 4월
- 계급: 일병 5호봉
- 소속: 제23교육연대 (논산)
- 병과: 보병
```

[ 이전으로 ] [ 가입 완료 ]
